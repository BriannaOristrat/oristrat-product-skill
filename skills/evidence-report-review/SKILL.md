# Evidence Report Review

Use this skill to review `01_运行过程证据报告.md/html/pdf`.

Follow `docs/checklists/01-evidence-report-review.md`.

## Minimum Checks

- Every conclusion can be traced to screenshots, JSON logs, API responses, or performance files.
- Formal load failures include `formal_load_failure_responses.json`.
- Evidence paths are relative or file names, not local absolute paths.
- PDF has no browser default header/footer path.

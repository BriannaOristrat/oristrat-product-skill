# 外部调研需求发现

## 适用条件

用户要求调研开源仓库、服务商、竞争产品、价格、免费试用、市场或外部事实时，在检索前使用本文件。

## 最小调研卡片

| 字段 | 必填内容 | 缺失时的动作 |
|---|---|---|
| 调研对象 | 仓库、服务商、技术路线或组合 | `NEED_CLARIFICATION` |
| 交付 | 清单、对比、短名单、PoC 或选型 | `NEED_CLARIFICATION` |
| 优先级 | 哪一类优先，是否都必须覆盖 | `NEED_CLARIFICATION` |
| 硬约束 | 许可、内网、地区、预算、不自训练等 | `NEED_CLARIFICATION` |
| 免费定义 | 免费类型、额度、期限或申请条件 | `NEED_CLARIFICATION` |
| 技术展开深度 | 一行用途或架构比较 | `NEED_CLARIFICATION` |
| 成功标准 | 每项候选必须具备的证据 | `NEED_CLARIFICATION` |

只询问会改变上述字段的 1–3 个问题；不要把卡片变成冗长问卷。

## 免费分类

| 标记 | 含义 |
|---|---|
| `FREE_FOREVER` | 可长期免费使用，限制明确。 |
| `FREE_QUOTA` | 有固定免费额度或资源上限。 |
| `TIME_LIMITED_TRIAL` | 明确的试用期限。 |
| `SALES_APPLIED_POC` | 需申请、审批或签约的 PoC；不得称为已确认免费。 |
| `UNKNOWN` | 未找到官方试用条款。 |

## 候选验收证据

| 对象 | 最少字段 |
|---|---|
| 开源仓库 | 官方仓库链接、许可证、部署入口、近期维护或发布证据、适用边界。 |
| 服务商 | 官方产品页、官方价格/试用页、免费分类、额度/期限、开通条件、数据边界。 |

技术组件仅填写“用途”和“适用边界”。用户未要求架构时，不展开模型、数据库、流程图或实现组合。

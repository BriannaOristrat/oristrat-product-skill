# 报告输出映射

## 三份报告的角色

| 报告 | 主要读者 | 内容粒度 | 是否客户可读 |
|---|---|---|---|
| `01_运行过程证据报告` | 内部复核、审计 | 细 | 不一定 |
| `02_实际交付报告` | 客户、业务、管理层 | 中 | 是 |
| `03_五轮业务论证与结果` | 产品、交付、管理层 | 中 | 是 |

## 证据来源映射

| 证据 | 进入 01 | 进入 02 | 进入 03 |
|---|---|---|---|
| 页面截图 | 文件索引和关键截图 | 仅摘要或必要图 | 不放或少量引用 |
| DOM / 接口响应 | 可放摘要和文件名 | 不放原始细节 | 不放 |
| 失败响应样本 | 必须展开 | 只放摘要和建议 | 只放结论 |
| 压测 summary | 必须展开 | 指标摘要 | 结论一致性 |
| 大数据造数/清理 | 必须展开 | 摘要 | 用例结论 |
| 缺陷/风险 | 证据和影响 | 风险摘要 | 业务影响 |

## 禁止映射

| 内容 | 处理方式 |
|---|---|
| 本机绝对路径 | 转为相对路径或文件名 |
| 浏览器页眉页脚 | 重新生成 PDF |
| 密码、手机号、租户码 | 脱敏 |
| Prompt、临时修复过程 | 不进入正式报告 |
| 错误版 PDF | 删除，不进入 zip |
